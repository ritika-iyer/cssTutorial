CSS Background Properties

The background properties are used to make the webpage's background look better.


CSS background-color

The background-color property specifies the background color of an element.

body {
background-color: violet;
}
h1 {
background-color: green;
}
div {
background-color: powderblue;
}
p {
background-color: yellow;
}

Example:

This element has a powderblue background color.



CSS background-image

The background-image property specifies an image to use as the background of an element. You should mention the name of the image file with extention or provide the image link address. By default, the image is repeated so it covers the entire element.

body {
background-image: url("iiit.jpg");
}

This has a background-image.


If you want to avoid repetition:

body {
background-image: url("iiit.jpg");
background-repeat: no-repeat;
}


CSS Background : Shorthand

Instead of writing down each property separately, you can use the shorthand property to write it all in a single line of code.

body {
background: #ffffff url("iiit.jpg") no-repeat;
}

CSS Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.

  • solid - gives a solid border
  • dotted - gives a dotted border
  • dashed - gives a dashed border
  • double -gives a double border
  • groove - gives a 3D grooved border; changes with border-color value
  • ridge - gives a 3D ridged border; changes with border-color value
  • inset - gives a 3D inset border; changes with border-color value
  • outset - gives a 3D outset border; changes with border-color value
  • none - gives no border
  • hidden - gives a hidden border.

p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

For Example:

This is a dotted border.

This is a double border.


You can also have mixed-style borders.

p {
border-style: dotted dashed solid double;
}

It will look like this:

Mixed-style border


CSS Border Width

The border-width property lets us decide the width of the border of any given p, div or the body itself.

p.one {
border-style: solid;
border-width: 5px;
}
p.two {
border-style: solid;
border-width: medium;
}

For example:

This has a double border with border-width of 10px.

This has a groove border with border-width medium.

You can also have side-specific border-width :

p{
border-style: solid;
border-width: 25px 10px 4px 35px;
}

This sets individual side-specific borders with widths 25px top, 10px right, 4px bottom and 35px left.

CSS border-color

You can use border-color property to make the borders colourful.
You can also specify different colours to each side.

p {
border-style: solid;
border-color: teal;
}
p.individual {
border-style: solid;
border-color: pink green red yellow;
}

For Example:

This has border colour Rebecca Purple.


This has individual border colour for each side.


CSS border-radius

You can use border-radius property to make the borders more rounded.

p {
border-style: solid;
border-radius: 10px;
}

This has border-radius of 10 px.


CSS Borders : Shorthand

Instead of writing down each property separately, you can use the shorthand property to write it all in a single line of code.

p {
border: 3px dotted blue;
}

CSS Padding

The padding property generates space around an element's content, inside of any defined borders.


Individual Padding

This property specifies the individual padding of each side of an element.

div {
padding-top: 50px;
padding-right: 30px;
padding-bottom: 50px;
padding-left: 80px;
}


CSS Padding Shorthand

You can use padding shorthand property to specify padding in a single line of code.

div {
padding: 25px 50px 75px 100px;
}

Or you could simply mention one value, like padding: 50px; and that would set all sides to the same amount of padding.
For example:

This has 20 px padding.

CSS Margin

The CSS margin properties are used to create space around elements, outside of any defined borders.


Individual Margin

This property specifies the individual margin of each side of an element.

div {
margin-top: 50px;
margin-right: 30px;
margin-bottom: 50px;
margin-left: 80px;
}


CSS Margin Shorthand

You can use padding shorthand property to specify padding in a single line of code.

div {
margin: 75px 50px 45px 130px;
}

Or you could simply mention one value, like margin: 50px; and that would set all sides to the same amount of margin.
For example:

This has 20 px margin.
