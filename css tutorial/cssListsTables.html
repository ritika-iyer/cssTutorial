Lists

In HTML, there are three types of lists. Two of the main types that are most commonly used are:

  • unordered lists <ul> where the list items are marked with bullets.
  • ordered lists <ol> where the list items are marked with either numbers or letters

The third type is definition lists that will not be discussed here.


The CSS list properties allows one to:

  • Set different list item markers for ordered and unordered lists
  • Set an image as the list item marker
  • Add background colors to lists and list items

Different Lists Item Markers

The list-style-type property specifies the type of list item marker.

For example:

<ul style:"list-style-type: cirlce;">
<li>Killua Zoldyck</li>
<li>Gon Freecs</li>
</ul>
<ol style="list-style-type: upper-roman;">
<li>Miya Atsumu</li>
<li>Miya Osamu</li>
</ol>


Output:

unordered list:

  • Killua Zoldyck
  • Gon Freecs

ordered list:

  1. Miya Atsumu
  2. Miya Osamu

Image as The List Item Marker

The list-style-image property specifies an image as the list item marker.

For example:

<ul style="list-style-image: url('https://bit.ly/3asTkV9')">
<li>inarizaki</li>
<li>itachiyama</li>
</ul>


Output:

  • inarizaki
  • itachiyama

List - Shorthand property

The list-style-image property is a shorthand property that is used to set all the list properties in one declaration.

For Example:

ul {list-style: square inside url("https://bit.ly/3asTkV9");}


When using the shorthand property, the order of the property values are:

  • list-style-type (if this is specified, the value of this property will be displayed if the image id unable to be displayed)
  • list-style-position (it specifies whether the list-item markers should appear inside or outside the content flow)
  • list-style-image

If any one of the property values above are missing, the default value for the missing property will be inserted, if any.


Styling List With Colors

One can even style the lists using colors to make it look as desired. Anything that is added to the <ol> and <ul> tags affects the whole list while properties added to the <li> tag affects the individual list items.

For example:

<ol style="background-color:#302827; padding: 20px;">
<li style="background-color:#E5B2FF;">derse</li>
<li style="background-color:#FFFEB2;">prospit</li>
</ol>

Output:

  1. derse
  2. prospit

Tables

CSS can be used to greatly improve the design of tables made with HTML.


Table Borders

The border property is used to specify the borders of a table in CSS.

For Example:

table, th, td {border: 1px solid black;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

The above example has double borders because both the table and the <th> and <td> elements have seperate borders.


Collapsing Borders

The border-collapse property sets whether the table borders should be collapsed into a single border

For Example:

table {border-collapse: collapse;}
table, th, td {border: 1px solid black;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

If a user only requires one border that is around the table, then they only need to specify the border property for <table>

For Example:

table {border: 1px solid black;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

Table Height and Width

The width and height of a table are determined by the width and height properties.

For Example:

table {width: 100%;}
th {height: 55px;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

Table Padding

The padding property is used to control the space between the border and content in the table on the <th> and <td> elements.

For Example:

th, td { padding: 15px; text-align: left;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

Horizontal Dividers

The border-bottom property adds horizontal dividers to the <th> and <td> elements.

For Example:

th, td {border-bottom: 1px solid #D6012B;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

Table Color

The background-color property specifies the background color of the element it is added to while the color property specifies the color of the text in that element.

For Example:

th {background-color: #71FFAF; color: black;} td {background-color: #71EFFF; color: black;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu

Veritcal Alignment

The vertical-align property sets the vertical alignment (like top, bottom, or middle) of the content in <th> and <td>. By default, the vertical alignment of the content in a table is middle for both elements.

For Example:

td {height: 45px; vertical-align: bottom;}

Output:

Last Name First Name
Miya Atsumu
Miya Osamu
