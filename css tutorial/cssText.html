CSS Text Properties

Let's look at some text properties.


CSS Text Color

The color property specifies the color of the text in an element.

body {
color: white;
}
p {
color: pink;
}

Example:

This text is powderblue in colour.


Text Alignment

A text can be left or right aligned, centered, or justified. We use text-align property for this.

h1 {
text-align: center;
}
h2 {
text-align: left;
}
h3 {
text-align: right;
}


Example:

This text is in center.


Text Decoration

The text-decoration property is used to set or remove decorations from text.

a {
text-decoration: none;
}
h1 {
text-decoration: overline;
}
h2 {
text-decoration: line-through;
}
h3 {
text-decoration: underline;
}

Example:

This has a line through.


Text Transformation

The text-transform property is used to make text uppercase or lowercase, or capitalize the first letter of each word.

p.uppercase {
text-transform: uppercase;
}
p.lowercase {
text-transform: lowercase;
}
p.capitalize {
text-transform: capitalize;
}

Example:

This is in uppercase.


Text Indentation

The text-indent property is used to specify the indentation of the first line of a text.

p {
text-indent: 50px;
}

Example:

This text has 70px indentation.


Letter Spacing And Word Spacing

This is used to manage space between letters and words.

h1 {
letter-spacing: 3px;
}
h2 {
word-spacing: 10px;
}

Example:

This has letter spacing of 2px.

This has word-spacing of 15px.


Text Shadow

The text-shadow property is used to give a shadow effect to the text.
You can specify the horizontal shadow, the vertical shadow and also the color of shadow.

h1 {
text-shadow: 2px 2px pink;
}

Example:

This has text-shadow effect with 3px shadow of purple color.


CSS Fonts

The CSS Fonts let you decide the font properties of your text.

CSS Font Family

This property specifies what kind of font the text should be in.
These include Times New Roman, Georgia, Lucida Console, Arial,etc.

p{
font-family: Arial;
}

Example:

This is in Lucida Console Font.


CSS Font Style

The font-style property is used to make text italic,normal or oblique.

p.one {
font-style: normal;
}
p.two {
font-style: italic;
}
p.three {
font-style: oblique;
}

Example:

This text in italics.


CSS Font Size

The font-size property sets the size of the text.It can be done using px, percentage (%), em or keywords like large, larger, small, etc.

h1 {
font-size: 40px;
}
h2 {
font-size: 1.875em;
}
body {
font-size: 100%;
}

Example:

This has a font size of 30px.


CSS Font weight

The font-weight property sets the weight/thickness of the text. We can use numbers as well as keywords like bold, bolder, loihter, etc.

h1 {
font-weight: 40px;
}

Example:

This text has font-weight "bolder".
