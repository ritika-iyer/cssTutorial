Introduction to CSS

CSS stands for - Cascading Style Sheets. It is used to make the webpage more appealing to the viewer.
It is used to style the backgrounds, borders, texts, spacing, colors, images, etc. on a webpage.

CSS Syntax

A CSS rule-set consists of a selector and a declaration block:

p { color: red;}


Here p is the selector, color is the css property and red is the css property value.

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

#paragraph1{ color: orange;}


CSS can be implemented in 3 ways. They are:
  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

Inline CSS means applying a CSS style to an individual element, inside of its HTML tag, in an HTML document. Style rules applied with inline CSS will override any styles from an internal or external style sheet.

It can be implemented as shown below :


<tag style = "color: red;"> Some Text </tag>

Internal CSS

The internal CSS code is the one that goes inside the head section of the HTML document. This is very useful when we want to give a specific page, its own custom style. There is no difference between the CSS code that you place inside an external file and the internal one. The only thing to remember is that we need to wrap it up inside the style tag.

It can be implemented as shown below :

<head>
  <style>
    p {
      color: blue;
      }
   body {
      background-color: grey
      }
 </style>
</head>

<body>
  <p> This is a paragraph in blue color with grey background </p>
</body>

External CSS

External CSS is most preferred way of implementing the CSS in the websites. With an external style sheet, you can change the look of an entire website by changing just one file.
Each HTML page must include a reference to the external style sheet file inside the element, inside the head section for its proper implementation.

It can be implemented as shown below :

HTML FILE: index.html

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>

<h1>Heading using h1 tag.</h1>
<p>Paragraph using p tag.</p>

</body>
</html>


EXTERNAL CSS FILE: style.css

body {
  background-color: teal;
  padding: 20px;
}

h1 {
  color: beige;
  margin: 15px;
}
