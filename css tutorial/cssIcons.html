Icons

In computing, an icon is a tiny picture displayed on a computer screen in order to help the user navigate a computer system.


How to Add an Icon

The simplest way to add icons to your webpage is through icon libraries such as 'Font Awesome'. Icons are scalable vectors that can customized with the help of CSS; for example their size, color, shadows etc.

Add the name of the specific icon class to any inline HTML element like <i> or <span>


Font Awesome Icons

for one to be able to use this icon library, they must visit font awesome and sign in to be able to get the code to add in the <head> section of their HTML page.

<script src="https://kit.fontawesome.com/yourcode.js"></script>


For Example:

<i class="fas fa-cloud" style="color: #C0D0D8;"></i>
<i class="fas fa-heart" style="font-size:24px;color:pink;"></i>
<i class="fas fa-car" style="font-size:48px;color:red;"></i>
<i class="fas fa-sync-alt fa-spin"></i>
<i class="fas fa-bars"></i>

Links


One can use CSS properties to style links in different ways to make them look more appealing on your webpage.


Styling Links

they can be styled with any CSS property such as color, background-color, font-size, etc.

For Example:

a {color: #999999;}


Output:

this is a link


In addition, links can be styled differently depending on what state they are in.

The four links states are:

  • a:link - a normal, unvisited link
  • a:visited - a link the user has visited
  • a:hover - a link when the user hovers their mouse over it
  • a:active - a link the moment it is clicked

For Example:

a:link {color: #DA1800;}
a:visited {color: #88FF00;}
a:hover {color: #47FFBE;}
a:active {color: #430D61;}

The order of setting styles when one uses several link states are:

  • a:hover MUST come after a:link and a:visited
  • a:active MUST come after a:hover

Link Text Decoration Property

The text-decoration as the name implies, is used to decorate the link text. In most occasions it is used to remove the underline from links.

For Example:

a {text-decoration: none;}


Output:

this is a link


Background Color

The background-color property can be used to specify a background color for links.

For Example:

a:link {background-color: #2FFF80;}
a:visited {background-color: #0A33AA;}
a:hover {background-color: #9469A8;}
a:active {background-color: #D80505;}


Link Buttons

Creating link buttons are possible by combining several CSS properties together to ultimately display links as boxes or buttons

For Example:

a {
background-color: #42D6C5;
color: white;
padding: 14px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}


Output:

this is a link
