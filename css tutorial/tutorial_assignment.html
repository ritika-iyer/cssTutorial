Assignment 1 :
Placement Cell

Suppose you are the Head of the Placement committee of your college. Several companies have arrived to provide the placement opportunities to the students. As the head, it is your duty to inform the students about the same through an official placement cell website. You can provide different pages for different branches too

Your website should contain the following:

  1. At least 3 divisions.
  2. A table displaying necessary information like company name, prerequisites, Package offered, test date, time, venue etc.
  3. Create an HTML form where students can register for the companies they are interested in, using radio button or dropdown. Provide other fields which you find necessary.
  4. Link the form to your Main webpage.
  5. Provide contact information.
  6. Use CSS to make your page look better.

Assignment 2 :
Advaita

Suppose you are the Head of the Managing Committee of Advaita. As the head, make a website informing people about the various events taking place during the 4-day college fest. You can create different pages for cultural and tech events.

Your website should contain the following:

  1. At least 3 divisions.
  2. A table displaying necessary information like event name, date, time and venue.
  3. Create an HTML form where students can register for various events. Provide various input types according to the field.
  4. Link the form to your Main webpage.
  5. Provide contact information.
  6. Use CSS to make your page look better.
