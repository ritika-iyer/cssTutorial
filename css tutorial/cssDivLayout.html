CSS Division

the < div > tag is used to define a division or a section in an HTML document like text, images, header, footer, navigation bar, etc.
It can easily be manipulated using "class" or "id" attribute.

It can be used like this:

<html>
  <body>

    <div style="border: 2px blue; color: green; text-align: left;">
      <h1>This is how you implement a div tag<h1>
    </div>

  </body>
</html>

Header & Footer

In HTML5, we can use <header> and <footer> tags to define the header section and footer section of a document respectively.

<style>
  header {
    background-color: #f218b0;
  }
 footer {
    background-color: #f90493;
  }
</style>

<body>
  <header>
    <h1> This is header section </h1>
  </header>
  <footer>
    <h4> this is footer section </h4>
  </footer>
</body>
